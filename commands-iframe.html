Commands

This page contains information about common used @commands.

By clicking on the cell/blue word, it will automotically copied the command then, you can paste it in-game.

COMMANDS DESCRIPTION
@refresh Refresh your screen
@autovend Auto vend for Merchant & Super Novice Class
@iteminfo Search for item information using item id or item name
Example: @iteminfo poring_card or @iteminfo 4001

Shortcut for this command is @ii
@noask Auto reject deal, any invite, and whisper from other player
@storage Access storage without going to kafra
@breakguild Disband your guild (must be the gm of the guild)
@mapmove Teleport your character based on the coordinate you entered
Example: @mapmove prontera 150 150

Shortcut for this command is @warp or @rura
@go Teleport your chacter on designated map using map id or map name (type the command to see the list)
Example: @go 0 or @go prontera
@servertime Show server date and time
@who Show all the players name on current map
@autoloottype Autoloot items based on type:
0 = healing items, 2 = usable, 3 = etc., 4 = armor, 5 = weapon, 6 = card, 7 = petegg, 8 = pet armor, 10 = ammunation
Example: @autoloottype 0

Shortcut for this command is @warp or @rura
@autolootitem Autoloot items based on item id or item name. Maximum of 10 items
Example: @autolootitem +4001 or @autolootitem +poring_card

To removed specific item on the list type @autolootitem -
Example: @autolootitem -4001 or @autolootitem -poring_card

Shortcut for this command is @alootid

To reset your list type @alootid reset
@autoloot Autoloot all items based on %
Example: @autoloot 10 (automatically loot all items with 10% drop rate and below)
@whereis Find the map location of the searched monster using monster id or monster name
Example: @whereis poring
@whodrops List down the monsters from highest to lowest % of drop rate of the search item using item id or item name
Example: @whodrops 4001 or @whodrops poring_card
@hominfo Show homunculus information
@homstats Show homunculus status
@request Send private message to Game Master in game
@mobinfo Show monster information using monster id or monster name
Example: @mobinfo poring or @mobinfo 1002

Shortcut for this command is @mi
